# Cuy Plugin

A comprehensive Minecraft economy/survival plugin for Paper 1.20+.

**Author:** Pi3ro  
**Version:** 3.0-SNAPSHOT  

## Features
- Economy system (money, /sell, /shop)
- Teams with home, chat, invites
- Shards (secondary currency)
- Bounty system
- RTP (Random Teleport)
- AFK system
- Combat manager
- Leaderboards (NPC-based)
- Amethyst custom tools
- TPA/TPAHere/TPAuto
- Warps & Homes
- ClearLag
- Tablist & Chat formatting
- Lunar Client integration (rich presence, team view)

## Dependencies
- [Vault](https://www.spigotmc.org/resources/vault.34315/)
- [packetevents](https://github.com/retrooper/packetevents)

## Build
```bash
mvn clean package
```

## Configuration
Edit files in `src/main/resources/` and `src/main/resources/economy/` before building.
